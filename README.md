# mmdb-bootstrap
custom bootstrap theme for mmdb

**how to use**

be sure to include the following in your `bower.json` file:

```json
{
  "overrides" : {
    "bootstrap-sass" : {
      "main" : [ ]
    },
    "jquery" : {
      "main" : [ ]
    }
  }
}
```

otherwise unwanted files may get injected based on your use of `main-bower-files`...
